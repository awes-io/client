# NuxtAuth

## Install

```bash
$ yarn install
```

## Development

```bash
# Development server with hot reload
$ yarn watch

# Tests in watch mode
$ yarn test:watch
```

## Build for production

```bash
# Build project
$ yarn build

# Lint in auto-fix mode
$ yarn lint:fix
```

Ensure to write proper commit message according to [Git Commit convention](https://www.conventionalcommits.org/)
