## How to install `client` to your local machine

```bash
# clone the repo
git clone git@github.com:awes-io/client.git

# go to project dir
cd ./client

# install all packages
yarn install

# init the lerna
lerna bootstrap
```

## Examples

### Basic UI in `dev` mode
```bash
yarn dev:eg:basic-ui
```
